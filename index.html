About Me

Hi ! I am a data science graduate with a background in engineering and business analytics. I excel at putting text data to use by using computational linguistics and machine learning techniques. I love creating end to end projects by curating data from interesting sources and extracting insights from them. I have also created mutliple proof of concepts for new products/services to showcase the marvels of artificial intelligence and how they can be put to use.

Experience

CRIM Research Center Montreal

Capstone Researcher

Developed a new text dissimilarity based framework for concordancer / keyword in context tool. This tool uses pre-trained word embeddings, elastic search and unsupervised dimensionalty reduction and clustering to retrieve the most semantically diverse instances of a search word in a given corpus. The tool was supported by a custom python package to help integrate the project with CRIM's existing code base.
